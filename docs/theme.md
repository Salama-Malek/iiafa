# Theme & Design Tokens

## Core Brand Palette
- `--primary-solid`: `#a97c50` (Clay Gold — primary CTAs, active indicators, highlights)
- `--primary-hover`: `#966d45` (Clay hover)
- `--primary-strong`: `#7f5c3a` (Clay pressed)
- `--primary-soft`: `#d7bea2` (soft clay tints)
- `--secondary-solid`: `#99141e` (Burgundy — secondary brand accents/headings/hero blending)
- `--secondary-hover`: `#7f1119`

## Semantic Colors
- `--danger`: `#99141e` (error/destructive)
- `--danger-strong`: `#7a1018`
- `--success`: `#2f7d4a`
- `--warning`: `#b3741f`
- `--info`: `#26629b`

## Neutrals & Surfaces
- `--ink`: `#1f1a16`
- `--muted-ink`: `#6f6256`
- `--surface-cream`: `#faf6f0`
- `--surface-soft`: `#f3eadf`
- `--surface-card`: `#fffcf7`
- `--surface-dark`: `#3f3d3d`
- `--surface-darker`: `#1e1d1d`
- `--border-warm`: `#dfd2c3`

## Usage Rules
- Primary actions use clay (`bg-primary`).
- Burgundy is used for secondary brand accents, selected headings, and deep gradient blending.
- Errors/destructive continue to use burgundy semantic destructive tokens for consistency.
- Focus rings remain clay for visibility on light/cream surfaces.
